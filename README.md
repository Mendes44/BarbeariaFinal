# 💈 Landing Page - Barbearia Imperial

Landing page desenvolvida para um comércio local hipotético, com foco em uma **barbearia fictícia**.  
O projeto foi criado como prática de **HTML, CSS e JavaScript**, com uma proposta visual moderna, navegação interna e estrutura organizada.

---

## 📌 Sobre o projeto

A ideia deste projeto é simular a página institucional de uma barbearia, apresentando os principais elementos que um pequeno negócio local costuma precisar em sua presença online:

- identidade visual no topo
- menu com navegação interna
- banner de destaque
- apresentação de serviços
- seção institucional
- mapa de localização
- contatos rápidos
- pequenos efeitos visuais para melhorar a experiência

---

## ✨ Funcionalidades

- Logo no topo da página
- Navegação por âncoras entre seções
- Banner principal com chamada visual
- Seção de serviços
- Seção "Sobre Nós" com imagem
- Seção de contato com Google Maps incorporado
- Alternância de cores entre seções
- Efeito visual no header ao rolar a página
- Animação suave de entrada dos elementos no scroll
- Estrutura separada em arquivos organizados

---

## 🛠 Tecnologias utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **Google Maps Embed**
